# Image-classification_Cifar10
Cifar10图片数据集分类项目

1. Cifar10数据集文件里有下载地址（点击readme.html），运行readcifar10.py解析出图片，建立TEST和TRAIN文件夹 保存图片数据
2. 运行train.py开始训练
